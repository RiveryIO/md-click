
# namer full

A CLI that gets name and last name and returns the full name

## Usage

```
Usage: namer full [OPTIONS]
```

## Options
* `name` (REQUIRED): 
  * Type: STRING 
  * Default: `none`
  * Usage: `--name`

  The user name


* `lastname`: 
  * Type: STRING 
  * Default: `none`
  * Usage: `--lastName`

  The last Name


* `help`: 
  * Type: BOOL 
  * Default: `false`
  * Usage: `--help`

  Show this message and exit.
